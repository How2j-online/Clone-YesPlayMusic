## YesPlayMusic Clone Docs

### 1. 项目结构

```text
Clone
├─ docs  // 文档目录
├─ public  // 静态资源
│    ├─ favicon.ico
│    ├─ img  // 图片文件夹
│    └─ robots.txt  // 爬虫文件
├─ src  // 源码目录
│    ├─ App.vue  // 根组件
│    ├─ assets  // 资源文件夹
│    │    ├─ css  // 样式文件夹
│    │    ├─ fonts  // 字体文件夹
│    │    └─ icons  // 图标文件夹    
│    ├─ components   // 组件文件夹
│    ├─ hooks  // 自定义hook文件夹
│    ├─ locales  // 国际化文件夹
│    ├─ router  // 路由文件夹
│    ├─ store  // vuex文件夹
│    ├─ utils  // 工具文件夹
│    └─ views  // 页面文件夹


├─ .commitlintrc.json
├─ .cz-config.json
└─ types  // 类型文件夹
```
